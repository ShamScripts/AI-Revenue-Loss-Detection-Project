# Results directory (convention)

Publication-style metrics and ablations are stored as CSVs under **`processed_data/`**, for example:

- `report_table_1_ieee_cis.csv`, `report_table_2_elliptic.csv`, `report_table_3_ablation.csv`
- `final_hybrid_comparison_metrics.csv`, `final_hybrid_threshold.txt`
- `elliptic_graph_experiments.csv`

Figures referenced in the report and Streamlit app are under **`figures/`**.
